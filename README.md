🚲 Bike Buyers Analysis – Excel Dashboard Project

Overview
This project analyzes customer data to understand which characteristics most strongly influence bike purchases. Using Excel, I cleaned the dataset, performed exploratory analysis, and built an interactive dashboard that allows users to quickly explore purchasing patterns across different customer segments.
________________________________________

🎯 Project Objective
Identify the key demographic and behavioral factors associated with bike purchases, including:
•	Income
•	Age and age groups
•	Commute distance
•	Gender and marital status
•	Education and occupation
•	Number of children, car ownership, and home ownership
•	Geographic region
________________________________________

📁 Project Contents
File	Description

🔍 Key Insights
Some of the main insights identified through the analysis include:
•	Customers with a 0–1 mile commute are significantly more likely to purchase bikes than those with longer commutes
•	The 30–55 age group accounts for the highest number of bike purchases
•	Bike buyers have a slightly higher average income than non-buyers (~$57.5k vs ~$55k)
•	Male bike buyers show the highest average income among customer segments
•	Very few bike purchases occur among customers aged 65 and above
All insights can be explored interactively within the dashboard.
